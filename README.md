## ¿Cómo podes levantar el proyecto ?

Nos posicionamos en la raíz del repositorio y ejecutamos el siguiente comando:

$ docker-compose up

Luego nos diriginos a nuestro navegador e ingresamos a la url:

http://localhost:8080
